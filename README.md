# student_mark_analyzer
This is a menu-driven C++ console application that stores and analyzes student academic records.  

It allows entering details such as Roll Number, Name, and Marks, and provides multiple operations like sorting, searching, and statistical analysis

## Features

| Functionality | Description |
|--------------|-------------|
| Input Data | Enter roll number, name, and marks for multiple students |
| Display | View the stored record of all students |
| Statistics | Calculates **Average, Maximum, and Minimum marks |
| Sorting | Sorts student records based on marks (Ascending / Descending) |
| Search by Roll No | Displays student information by roll number |
| Search by Name | Displays student information by name |
